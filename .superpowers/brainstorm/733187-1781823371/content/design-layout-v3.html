Section 3: Both Paths Work the Same Way

🏠 Local Deployment

  1. Scaffold tracker files in local folder
  2. Install systemd service
  3. Start tracker on specified port
  4. Verify health — 200 OK
  5. Save project & show Open Tracker button

☁️ Remote Deployment

  1. Test SSH connection first
  2. Scaffold tracker files (temp local)
  3. SSH: create remote folder
  4. SCP files to remote server
  5. SSH: install & start systemd service
  6. Verify remote health — 200 OK
  7. Save project & show Open Tracker button
